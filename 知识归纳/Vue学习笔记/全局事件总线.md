全局事件总线：任意组件间通信

```js
main.js

	new Vue({
		...
		beforeCreate() {
			Vue.prototype.$bus = this;    // 安装全局事件总线，$bus就是当前应用的Vue实例（vm）
		}
		...
	});
```

```js
[Vue Component].vue

	...
	mounted() {
		this.$bus.$emit('[event name]', data);    // 传递数据并触发事件监听
		this.$bus.$on('[event name]', (data) {});    // 监听事件触发
	},
	beforeDestroy() {
		this.$bus.$off('[event name]');    // 销毁监听事件
	}
```
